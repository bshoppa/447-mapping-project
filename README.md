# 447-mapping-project

## System requirements

- Python3 installed
- Able to use the virtual environment on python3
- All python packages are included

to run, run this command (linux):

```sh
./start.sh
```

or this command (windows)
```sh
447proj\scripts\activate
flask run
```

to view the app while it's running

go to

localhost:5000

or 127.0.0.1:5000
